## Project Overview  
A **production-grade, CI/CD-ready** test automation suite that validates a full **login → add to cart → checkout** flow in a custom **ASP.NET Core e-commerce web app**.

- **Zero manual setup** — Uses **Selenium Manager** to auto-download ChromeDriver  
- **No hardcoding** — Works on any machine with Chrome 142+  
- **Real browser validation** — Visible Chrome (or headless in CI)  
- **100% pass rate** — Explicit waits, alert handling, visibility checks  
- **Live UAT simulation** — Interactive UI with JavaScript feedback
